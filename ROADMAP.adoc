= Burble Roadmap

== Current Status

All MUST and SHOULD features are implemented. 193 tests, 0 failures.
All in-code TODOs resolved (2026-03-29): Bebop VoiceSignal decode for
tags 6-8, safe callback registration with typed NifEvent polling.

== Milestones

=== v0.1.0-alpha.1 — Foundation (DONE)

* [x] WebRTC voice engine (SFU, Opus, ExWebRTC)
* [x] Push-to-talk and VAD mode switching
* [x] E2EE via Insertable Streams (X25519 + AES-256-GCM)
* [x] Room management (create, join, leave, permissions)
* [x] Text chat (NNTPS threading, Phoenix channels)
* [x] Auth (register, login, guest, magic link, JWT)
* [x] Self-test diagnostics (16 tests, 3 modes)
* [x] Zig SIMD coprocessor (audio, DSP, neural, compression, crypto)
* [x] VeriSimDB integration (replaces PostgreSQL)
* [x] Containerised deployment (Chainguard, port 4020)

=== v0.2.0 — Interop Bridges (DONE)

* [x] Discord bridge (Gateway + Voice WS + UDP, xsalsa20_poly1305)
* [x] Mumble bridge (TCP protobuf + UDP Opus, TLS)
* [x] Matrix bridge (Client-Server API, /sync long-poll, m.call.*)
* [x] SIP gateway (INVITE/ACK/BYE, RTP, G.711↔Opus transcoding)

=== v0.3.0 — Transport Layer (DONE)

* [x] QUIC transport (0-RTT, multiplexed streams, datagram voice)
* [x] RTSP broadcast module
* [x] Multipath redundancy (3-port UDP striping)
* [x] Bebop wire protocol schemas (voice_signal.bop, room_event.bop)
* [x] QUIC Bebop signal dispatch (VoiceSignal union decoding)

=== v0.4.0 — Integration (DONE)

* [x] Groove discovery endpoint (/.well-known/groove)
* [x] IDApTIK voice adapter (spatial audio, stealth whisper)
* [x] PanLL workspace huddle engine
* [x] Gossamer desktop client hooks (tray, hotkeys, PipeWire)
* [x] Extension API (core/profiles/extensions three-tier)

=== v1.0.0 — Stable Release (DONE)

* [x] QUIC hardening (rate limiting, datagram size enforcement, ticket rotation)
* [x] Bebop codegen (Mix task + generated Burble.Protocol.{VoiceSignal,RoomEvent})
* [x] Idris2 ABI formal proofs (Avow consent, Vext integrity, Permissions hierarchy)
* [x] QUIC signal handler refactored to use generated Bebop decoders
* [x] Chainguard base image pinned by digest
* [x] Version bump to 1.0.0 across mix.exs, Justfile, groove manifest
* [ ] Zig FFI bridge from BEAM to Idris2 ABI
